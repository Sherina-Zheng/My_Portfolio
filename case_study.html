Plant Health API

Duration: 08/27/2024-10/31/2024

Overview:

The Plant Health Identifier API helps diagnose plant diseases and health issues in crops and houseplants. As many first-time house plant growers like I, one struggles with early detection, leading to reduced yields. This API trial project analyzes plant images to identify diseases, deficiencies, and pests, providing real-time insights and treatment recommendations.

Project Analysis: Challenges Faced vs. Key solutions implemented to resolve problems -

Challenges:

- Pictures taken with bad quality or angles may impact the accuracy of the results or even false identification

- Some API’s databases may not be broad engough to include all plant species

- Variations of a plant that is significant to a specific location or region may involve additional research to obtain accuracy

Solutions:

- Limit the pixles to a specific index for users and pop an alert if bad angles are detected on the pictures that are uploaded

- Fetch API keys from various API open resources and use Machine Learning integration

Summary:

This API improves plant health monitoring with fast, intelligent detection. Early tests show 90%+ accuracy in disease identification. Users can take proactive plant care steps, reducing disease spread and boosting yield. By combining image recognition and symptom-based diagnostics, it makes plant care more efficient and accessible.

Insect Identifier

Duration: 08/27/2024-10/31/2024

Overview:

The Insect Identifier API helps users identify insect species from images, supporting biodiversity studies, pest control, and educational purposes.

Project Analysis:

- API Integration: Leveraged an existing image recognition API (e.g., Microsoft Azure, Clarifai) with an API key for insect identification.

- Data Setup: Compiled a database of insect species with details like habitat and behavior, linked to the API output.

- User Interface: Developed a basic interface for users to upload images and receive identification results.

- Additional Features: Included ecological information from online databases to enrich the results.

Summary:

The API provides quick and accurate insect identification, making it useful for researchers and hobbyists. Future improvements could focus on adding more species and refining the user interface.

Mushroom Identifier

Duration: 08/27/24-10/31/24

Overview:

Mushroom Identifier API helps users identify mushroom species from images, providing details like edibility and habitat. It’s designed to assist foragers and hobbyists in safely identifying mushrooms.

Project Analysis:

- API Integration: Used an existing image recognition API (e.g., Google Vision, AWS Rekognition) with an API key for species identification.

- Data Setup: Created a custom database of mushroom species with details like edibility and habitat, linked to the API output.

- User Interface: Built a simple web interface for users to upload images and view results.

- Safety Features: Added warnings for toxic species based on the API’s identification results.

Summary:

API successfully identifies common mushroom species and provides useful information for foragers. Future work could include expanding the database and improving accuracy for rare species.
